# <スタンプ屋さん>

## サイト概要
### サイトテーマ
LINEなどのSNSで使えるスタンプを紹介する場所。作成者と使用者が関われるコミュニティーサイト

### テーマを選んだ理由
知り合いがLINEスタンプを作っていたので、それを紹介できる専用サイトを作ってみようと思ったのでこのテーマにしました。

### ターゲットユーザ
LINEなどのSNSで使えるスタンプを作っている方と、スタンプを使おうとしている方

### 主な利用シーン
作成者側は、作ったスタンプを誰かに使って欲しい人。
使用者側は、スタンプの作成者にコメントを送れる
